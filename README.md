# MedRAG AI: An Intelligent Research Assistant for Medical AI Papers

## Overview

MedRAG AI is an end-to-end Retrieval-Augmented Generation (RAG) application developed as the final project for the DataTalksClub LLM Zoomcamp 2026.

The application enables users to interact with a curated collection of Medical AI research papers using natural language. Instead of manually searching through dozens of publications, users can ask questions and receive grounded answers supported by retrieved document passages.

## Features

- PDF document ingestion
- Automatic text extraction
- Document chunking
- Sentence embeddings
- ChromaDB vector database
- Hybrid retrieval (BM25 + Vector Search)
- Document re-ranking
- Query rewriting
- GPT-powered answer generation
- Streamlit user interface
- Retrieval and LLM evaluation
- Monitoring dashboard
- Docker support

## Technology Stack

- Python 3.12
- Streamlit
- ChromaDB
- sentence-transformers
- OpenAI GPT-5
- PyMuPDF
- Plotly
- SQLite
